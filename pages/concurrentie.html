Concurrentie analyse

Band nr. 1 : De Party Band

Op het eerste zicht ziet de website van De Party Band er een beetje chaotisch uit.

Pluspunten:

  • Dynamische What’s New-banner
  • Interactieve linken naar komende events
  • Like-button gelinkt met facebook

Minpunten:

  • Nutteloze pagina’s
  • Onleesbare tekst
  • Je kan de band enkel inhuren na een telefoontje, dus optie om via de website contact op te nemen.

Band nr. 2 : Diaz

Pluspunten:

  • Dynamische lay-out met bewegende onderdelen
  • Je kan ver scrollen op de homepagina
  • Links naar youtube-video’s
  • Links naar sociale media
  • Je kan direct chatten via de messenger functie
  • Booking knop blijft aanwezig in banner op elke pagina
  • Contactformulier

Minpunten:

  • Knop voor media pagina is geen bruikbare knop, maar een hoover voor twee andere pagina’s

Band nr. 3 : Trigger Finger

Deze website straalt meteen karakter uit.

Pluspunten:

  • Homepagina bevat geen cruciale informatie maar is een aandachtstrekker
  • Links naar alle sociale media duidelijk in beeld
  • Tickets staan te koop naast betreffende events
  • Muziek kan via de pagina meteen worden beluisterd via spotify, deezer en andere providers.
  • Meer beelden dan tekst
  • Booking-knop

Minpunten

  • Video op de achtergrond van de homepage maakt de pagina wel erg druk
